== Description ==

A simple plugin to make cookie info bar with link to page   

You can also remove the saved redirects if it get's old.

== Installation ==

1. Upload `nitea-cookie-info` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
